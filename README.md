# GraphDigitizer Pro - 专业图形数字化工具

> 从科学论文、技术报告中的图表图像精准提取数据点，比 GetData Graph Digitizer 更好用的开源替代品。

## ✨ 核心特性

### 🎯 精准取点

- **智能放大镜**：实时显示鼠标附近的放大区域，带十字线辅助
- **像素级精度**：坐标实时显示，精确到亚像素级别
- **右键取色**：随时拾取任意像素的颜色信息

### 📐 坐标系标定

- **两点标定法**：X轴两点 + Y轴两点，简单四步完成标定
- **线性/对数轴**：同时支持线性轴和对数轴
- **仿射变换**：支持倾斜坐标轴的精确变换

### 🔍 自动曲线追踪

- **颜色匹配追踪**：基于颜色自动提取整条曲线
- **可调参数**：颜色容差、采样间隔、线宽等参数可调
- **颜色拾取器**：点击图像直接拾取目标曲线颜色

### 📈 多曲线管理

- **独立管理**：每条曲线独立管理，支持不同颜色标识
- **可见性切换**：自由控制每条曲线的显示/隐藏
- **数据预览**：实时数据表格预览

### 💾 丰富的导出格式

- **CSV** - 逗号分隔值，通用表格格式
- **JSON** - 结构化数据格式
- **MATLAB (.m)** - 可直接运行的 MATLAB 脚本，包含绘图代码
- **Python (.py)** - 可直接运行的 Python 脚本，使用 matplotlib 绘图
- **剪贴板** - 一键复制，直接粘贴到 Excel

### 🔧 图像增强

- **亮度/对比度调节**：增强模糊图像的可读性
- **灰度模式**：去除颜色干扰，专注曲线识别
- **实时预览**：调节参数即时看到效果

### ⚡ 高效操作

- **完整的撤销/重做**：最多 100 步操作历史
- **拖放支持**：直接拖拽图像文件到窗口
- **全键盘快捷键**：常用操作一键完成
- **Ctrl+滚轮缩放**：流畅缩放体验
### 使用流程

1. **打开图像** (Ctrl+O)
   - 加载包含图表的 PNG/JPG/BMP 等图像

2. **标定坐标系** (按 1)
   - 点击 X 轴上一个已知值的位置，输入该值
   - 点击 X 轴上另一个已知值的位置，输入该值
   - 点击 Y 轴上一个已知值的位置，输入该值
   - 点击 Y 轴上另一个已知值的位置，输入该值

3. **提取数据** (按 2 或 3)
   - **手动取点**：在曲线上逐点点击提取
   - **自动追踪**：设置曲线颜色/容差，一键提取整条曲线

4. **导出数据**
   - 选择所需格式导出（CSV/JSON/MATLAB/Python）
